# How to release the Wikipedia Preview Wordpress plugin

* WP core
** up version
** publish
** use from WPWPP
* WPWPP
** up version
** get svn repo
** run publish script
** diff and commit svn repo
* Test
** plugin page
** update plugin with local
** test plugin features
* How to revert
** Update "Stable tag: <last known good version>" in svn and commit
* Announce on slack
